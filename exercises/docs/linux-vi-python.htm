Linux (Command Line) Commands

 

pwd                print working directory (where am I?)

ls                     list contents of current directory

cd                    change directory 

                        examples:     cd exercises  (change to exercises directory)

                                                cd ../ (move up one directory)

                                                cd ~/  (go to home directory from anywhere)

                                                cd ~/examples (go to exercises from anywhere)

PS1=’:)’          make your prompt an emoticon!

 

vi (Text Editor) Commands

 

To launch vi type ‘vi’ at command line after going to exercises directory.

 

vi has two modes:  ‘insert’ and ‘command’. 

 

Insert mode is what you use for typing.  The bar at the bottom will say ‘—INSERT---‘ if it is insert mode.  To get to insert mode, type ‘i’.  To escape insert mode, hit ESC.

 

Commands (in Command Mode)

i                                   toggle to insert mode (for typing)

Working with files:   

:e filename.py          edit (open) a file (filename.py is an example)

:ls                                show all files that are open

:sp                              split window horizontally

:vs                               split window vertically

:w filename.py          write (save) file.  You can also just type :w 

:wqa                           write all files and quit vi

 

 

If  a file needs to be saved, it will have a plus sign next to it in the window bar.

 

Very often, you will find that you have accidentally hit some weird command buttons.  The best way to fix this is to RUN AWAY!  Hit escape, then :wqa, which will write your files and exit.  Once you are back at the command prompt, enter vi to start over.  Or, you can close the whole terminal, open a new one, and make sure you use cd ~/exercises.

 

Copying and pasting text

shift+v                        select lines

y                                  copy (yank) selection

p                                  paste (put) selection

 

Opening a Terminal in within vi

To open the shell (command line), mouse to the window you want to run programs from and hit F3+Enter.  A prompt should appear.  You need to be in ‘—INSERT—‘ mode in order to type commands at the terminal.

 

python Commands

 

To open the python (interactive) shell, type ‘python’  this will open a python shell with a prompt ‘>>’.  You can now type commands and see what happens.

 

To run a python script, type ‘python filename.py’.  The teacher will show you some examples to run.   Some of the exercises will have you enter arguments, or numbers that the program will use, as part of the command line.  For example, a 2 argument python command will look like this:  ‘python filename.py arg1 arg2’.
